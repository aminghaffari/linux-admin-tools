# Linux Admin Tools

A lightweight Bash toolkit for Linux system administration, troubleshooting, monitoring, and health checks.

## Tools

- `system-health` - Check CPU, memory, disk, services, and overall system status
- `network-health` - Check interface, gateway, DNS, packet loss, routing, and connectivity
- `disk-health` - Check disk usage, inode usage, mount points, and large directories
- `log-health` - Summarize critical logs, kernel warnings, OOM events, and auth failures
- `security-health` - Basic Linux security posture checks

## Usage

Run directly:

```bash
./bin/system-health
./bin/network-health
./bin/disk-health
./bin/log-health
./bin/security-health
